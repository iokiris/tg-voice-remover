# Музыкальный Бот для Telegram

Этот проект представляет собой **Telegram-бота**, который использует **FastAPI** сервер для обработки аудиофайлов. Бот позволяет пользователям отправлять музыкальные треки и получать их версию без вокала. Для обработки данных используется связка технологий: **Go**, **FastAPI** и **RabbitMQ**.

## Описание

1. **Telegram-бот** принимает аудиофайлы от пользователей в формате MP3.
2. После получения файла, **FastAPI** сервер обрабатывает его и отправляет запрос на сервер, написанный на **Go**, который взаимодействует с **RabbitMQ**.
3. В свою очередь, **RabbitMQ** передаёт задачу на обработку (вырезание вокала) и результат возвращается пользователю в виде обработанного аудиофайла.

Процесс вырезания вокала осуществляется с использованием специализированных библиотек и технологий, которые позволяют разделить аудиотрек на отдельные компоненты (инструменты и вокал) и удалить последнего.

## Технологии

- **Go** — используется для обработки задач и взаимодействия с **RabbitMQ**.
- **FastAPI** — сервер для обработки запросов и взаимодействия с ботом.
- **RabbitMQ** — система обмена сообщениями для асинхронной обработки задач.
- **Telegram API** — интерфейс для взаимодействия с пользователями через Telegram-бота.
